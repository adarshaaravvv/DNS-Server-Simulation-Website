01

Client Query

The client application sends a DNS query over UDP (port 53) or TCP for large responses. The query contains the domain name and requested record type (A, AAAA, PTR, etc.).

02

Recursive Resolution

The recursive resolver checks its local cache. On a cache miss, it queries root servers → TLD servers → authoritative servers in a hierarchical chain.

03

Server Response

The authoritative DNS server looks up its zone file and returns the resource record. The response includes the IP address, TTL value, and record class.

04

Cache & Deliver

The resolver caches the result for the TTL duration and forwards the answer to the client. Reverse lookup (PTR) queries follow the same path in the in-addr.arpa zone.
